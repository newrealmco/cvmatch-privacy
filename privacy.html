CVMatch – Privacy Policy

Last updated: 2025-10-02

Overview

CVMatch is a Chrome extension that helps you tailor your CV to the specific LinkedIn job page you are viewing. It runs locally in your browser and, by default, does not send data to the developer.

Data we process

  • CV content (text or a Google Doc URL you provide)
  • Job description extracted from the LinkedIn job page you open
  • Settings you choose (provider, optional model name, API key, toggles)

CVMatch stores settings and the last result locally using chrome.storage. You can clear all stored data any time via “Clear all data”.

When data may leave your device

Only when you click Check match, the extension sends your CV text and the job description directly to the AI provider you choose, using your API key. This request goes from your browser to the provider; the developer does not receive, proxy, or store this data.

Supported providers may include OpenAI, Anthropic (Claude), Google (Gemini), Perplexity, and xAI (Grok). Your use of a provider is subject to that provider’s terms and privacy policy.

Permissions and purpose

  • Host permissions (linkedin.com/jobs) – read the job details on pages you open, to compute the match.
  • Scripting – inject a minimal content script to extract job text when you ask.
  • Side panel – show the UI (settings, progress, results).
  • Storage – save your local settings and last result.
  • Offscreen – reliably generate and download a .docx file.
  • Alarms / Notifications – optional progress polling and completion notices for long runs.

Children’s privacy

CVMatch is not intended for use by children under 13.

Security

  • All extension code is packaged; no remote code is loaded.
  • Requests to AI providers use HTTPS and your API key.
  • You can remove data at any time via “Clear all data” or by uninstalling the extension.

Changes

If we change this policy, we will update the “Last updated” date above. Material changes may also be noted in the extension release notes.
